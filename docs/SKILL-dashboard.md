# SKILL: 운영 대시보드 `/dashboard`

## 목적
창고 운영 현황을 한 화면에서 파악. 오늘의 KPI, 입출고 트렌드, 재고 알림을 시각화.

## 레이아웃 구성

### KPI 카드 (상단 4개)
| 카드 | 지표 | 단위 |
|------|------|------|
| 오늘 입고 | 건수 | 건 |
| 오늘 출고 | 건수 | 건 |
| 전체 재고 | SKU 수 / 수량 | 종 / EA |
| 처리 대기 | 미처리 작업 수 | 건 |

각 카드: 숫자 크게 + 전일 대비 증감 표시(▲▼ 퍼센트)

### 차트 영역 (중단)
- **좌측 (60%)**: 주간 입출고 추이 — 라인차트 (7일, 입고/출고 2개 라인)
- **우측 (40%)**: 재고 카테고리 분포 — 도넛차트 (품목 카테고리별 %)

### 하단 테이블/리스트
- **최근 입고 내역** (5건): 날짜, 품목명, 수량, 담당자, 상태
- **출고 대기 목록** (5건): 수주번호, 고객사, 품목수, 요청일, 우선순위

### 알림 패널 (우측 사이드 또는 카드)
- 재고 부족 알림 (안전재고 미달 품목)
- 입고 지연 알림

## Mock 데이터 예시
```ts
const kpi = {
  todayInbound: 24,
  todayOutbound: 31,
  totalSKU: 847,
  pendingTasks: 7,
}
// 주간 트렌드: 7일치 배열
// 도넛: ['전자부품', '생활용품', '식품', '의류', '기타']
```

## 디자인 포인트
- Recharts 사용, 커스텀 툴팁
- 카드 배경 #1e293b, 포인트 blue-500
- 알림은 amber/red 컬러로 시선 유도
